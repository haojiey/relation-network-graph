操作功能扇形图展示右键节点

操作功能都可以在以下Promise方法中添加自定义事件处理（例如：删除节点时需要处理接口）

操作功能时默认的功能不需要再调用接口后重新渲染（自定义功能看情况，需要渲染可以调用实例的update方法，graph.update()）

click: () => { return new Promise((resolve) => { //自定义过度操作 resolve(false) }) }

连线：两节点之间连接一条关系线，可自定义关系名称（在过度操作中）

删除：删除当前节点以及当前节点的关系

收起：收起当前节点的下级节点

隐藏：隐藏当前节点以及当前节点的关系

展开：展开当前节点的下级节点

解锁：当点击节点以及拖拽节点后会将节点固定在某一位置，通过释放来解除固定
